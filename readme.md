# Интернет‑магазин

## Описание
Проект интернет‑магазина на Django.

## Структура
- `online_store/` — настройки проекта
- `catalog/` — приложение с каталогом товаров

## Запуск
1. Создать виртуальное окружение: `python -m venv venv`
2. Активировать: `source venv/bin/activate` (Linux/macOS) или `venv\Scripts\activate` (Windows)
3. Установить зависимости: `pip install -r requirements.txt`
4. Запустить: `python manage.py runserver`

## Ветви Git
- `main` — стабильная версия
- `develop` — основная разработка
- `feature/*` — ветки для задач
